##  Requisitos Previos

Para ejecutar este laboratorio, necesitarás tener instaladas las siguientes herramientas:

1. **Visual Studio Code**: El editor de código base.
2. **Extensión Live Server**: Para crear un servidor local y visualizar el proyecto.

---

##  Instalación y Uso

Sigue estos pasos para configurar el entorno:

1. **Obtén el código:** Descarga este repositorio como un archivo .zip o clónalo usando Git.
2. **Abre el proyecto:** Ejecuta Visual Studio Code y abre la carpeta que acabas de descargar.
3. **Lanza el simulador:**
* Busca el archivo `index.html` en el explorador de archivos de la izquierda.
* Haz **clic derecho** sobre el archivo.
* Selecciona la opción **"Open with Live Server"**.



---

##  Controles de Navegación

Una vez que el sistema solar se cargue en tu navegador de confianza, utiliza las siguientes teclas para navegar por el espacio:

| Tecla | Acción |
| --- | --- |
| **W** | Mover hacia arriba |
| **S** | Mover hacia abajo |
| **A** | Mover hacia la izquierda |
| **D** | Mover hacia la derecha |
| **E** | Zoom + (Acercar) |
| **Q** | Zoom - (Alejar) |

---

##  Resultado Esperado

Al seguir las instrucciones, se abrirá automáticamente una ventana en tu navegador donde podrás observar el comportamiento y la estética del sistema solar en un entorno 3D/interactivo.















Las herramientas que vamos a usar serán, para el frontend, next js, por lo cual deben instalar node js y pnpm (ya no npm por los recientes problemas de ciberseguridad); para el backend y servicios vamos a usar Django, por lo que hay que instalar python y django, y sqlalchemy para el orm e interacciones con la BD, y de BD vamos a usar una relacional, Postgresql. Por último, vamos a estar usando Docker para utilizar contenedores por cada servicio y que sea más fácil desplegar en producción. necesito instalar todo es que pide ese texto dime como paso a paso
